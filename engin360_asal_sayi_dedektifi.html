Asal Sayı Dedektifi

Bir sayı gir. Sistem, bilgisayar gibi düşünerek bölenleri tek tek kontrol etsin ve sayının asal olup olmadığını açıklasın.

🔎 Sayıyı Kontrol Et

Asal sayı; 1'den büyük olup yalnızca 1'e ve kendisine tam bölünen sayıdır.

Akıllı kontrol:
Bir sayının asal olup olmadığını anlamak için tüm sayıları denemeye gerek yoktur.
Sadece 2'den √n'e kadar olan bölenleri kontrol etmek yeterlidir.
Sonuç burada görünecek.

🧠 Algoritma Akış Şeması

1. Sayı al: n
2. n ≤ 1 ise asal değildir.
3. n = 2 ise asaldır.
4. n çift ise asal değildir.
5. 3'ten √n'e kadar tek bölenleri dene.
6. Tam bölen bulunursa asal değildir.
7. Hiç bölen bulunmazsa asaldır.
Kodlama mantığı
function asalMi(n) {
  if (n <= 1) return false;
  if (n === 2) return true;
  if (n % 2 === 0) return false;
  for (let i = 3; i <= Math.sqrt(n); i += 2) {
    if (n % i === 0) return false;
  }
  return true;
}
